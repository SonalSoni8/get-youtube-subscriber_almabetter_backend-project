Get YouTube Subscribers

The APIs provided by this application include the following:

(Click on the below card to get details you want)

GET /subscribers

Provides an array containing all the subscribers stored in the database.

GET /subscribers/names

Provide a response containing an array of subscribers, where each subscriber object includes only two fields: 'name' and 'subscribedChannel'.

GET /subscribers/:id

Provide a response containing the details of a subscriber (represented as an object) associated with the provided ID.

Default it will take to id: "64d21cfbdefea137818389c1", you can change it accordingly

Note: If the ID does not correspond to any subscriber, respond with a status code of 400 along with an error message (message: error.message).
